# Core Features

## Season Management

- Create new seasons
- View season history
- Archive completed seasons
- Minimum 4 races per season requirement

## Race Tracking

- Record race results
- Support 2-9 racers per race
- Track finishing positions
- Assign points based on placement

## Leaderboard

- Display current season standings
- Show total points per racer
- Sort by points (descending)
- Display race participation count

## Points System

- Progressive scoring (1st place = most points)
- Configurable point values
- Consistent scoring across season

## Racer Management

- Add/remove racers
- Track racer statistics
- Historical performance data
