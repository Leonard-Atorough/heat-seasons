# User Flows

## Create New Season (Admin Only)

1. Navigate to seasons page
2. Click "New Season"
3. Enter season name
4. Set start date
5. Save season

## Record Race Results (Admin Only)

1. Select active season
2. Click "Add Race"
3. Enter race date
4. Add racers and positions (2-9 racers)
5. System calculates points
6. Save race results
7. Leaderboard updates automatically

## View Leaderboard (All Users)

1. Select season
2. View sorted standings
3. See racer names, points, races participated
4. Optional: filter/sort by different criteria

## Manage Racers (User Scoped - Basically User access to their own racer profile)

1. Navigate to racers page
2. Add new racer with name
3. View racer history
4. Deactivate racer (if needed)

## Complete Season (Admin Only)

1. Ensure minimum 4 races recorded
2. Mark season as complete
3. Archive season
4. View final standings
